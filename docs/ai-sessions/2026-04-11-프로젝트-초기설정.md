infrastructure 모듈 구조:
```
infrastructure/{모듈명}/
├── interface/
│   └── {모듈명}.interface.ts
├── component/
│   └── {모듈명}.client.ts
└── {모듈명}.module.ts
```

## AI 응답 요약

### 최종 아키텍처
```
application → domain → infrastructure
```
- `infrastructure`: 순수 기술 클라이언트 (비즈니스 로직 없음)
- `domain`: 비즈니스 모듈 (infrastructure 사용)
- `application`: 유스케이스 (controller + service)

### 기술 스택 결정
- **런타임**: Node.js + TypeScript
- **HTTP 프레임워크**: Express
- **LLM**: Claude API (Anthropic SDK)

### 파일 네이밍 컨벤션
- `xxx.controller.ts` - 컨트롤러
- `xxx.service.ts` - 서비스
- `xxx.client.ts` - 기술 클라이언트
- `xxx.interface.ts` - 인터페이스
- `xxx.module.ts` - 모듈
